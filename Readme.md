MyopicVicar-Vagrant
===================

Vagrantfile for MyopicVicar

Installation
------------

    $ git clone https://github.com/justincy/MyopicVicar-Vagrant.git
    $ cd MyopicVicar-Vagrant
    $ vagrant up
	$ vagrant ssh
	$ cd MyopicVicar
	$ bundle install
	$ bundle exec rake db:migrate


Requirements
------------

* Virtualbox
* Vagrant
